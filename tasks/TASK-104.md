# TASK-104 - Add MAC vendor visibility toggle in devices toolbar

## Опис задачі

Після реалізації TASK-103 поле `mac_vendor` відображається під `Hostname` дрібним сірим текстом.

Необхідно додати в панель керування над таблицею нову кнопку:

`MAC`

Кнопка повинна керувати видимістю MAC vendor у таблиці пристроїв.

---

## Ціль задачі

Дати оператору можливість швидко вмикати або вимикати відображення `mac_vendor`, щоб таблиця залишалась компактною, коли vendor-інформація не потрібна.

---

## Поточна поведінка

Зараз `mac_vendor` відображається під hostname автоматично, якщо:

- `mac_vendor != null`
- `is_random_mac != true`

Приклад:

```text
IPHONE-okudr
Apple, Inc.
```

---

## Нова поведінка

Додати кнопку `MAC` у верхню панель керування таблицею, поруч з існуючими кнопками `End`, `All`, refresh, auto toggle або іншими control buttons.

### Якщо кнопка `MAC` активна

Показувати `mac_vendor` під hostname.

```text
IPHONE-okudr
Apple, Inc.
```

### Якщо кнопка `MAC` неактивна

Не показувати `mac_vendor`.

```text
IPHONE-okudr
```

---

## UI правила

### Кнопка

- Назва: `MAC`
- Тип: toggle button
- Візуально має відповідати існуючим кнопкам панелі
- Активний стан має бути помітним
- Неактивний стан має бути візуально нейтральним

---

## Логіка відображення vendor

Vendor показується тільки якщо одночасно виконані умови:

```text
showMacVendor == true
mac_vendor != null
is_random_mac != true
```

Якщо хоча б одна умова не виконана - vendor не показується.

---

## Поведінка для random MAC

Для пристроїв з:

```json
"is_random_mac": true
```

`mac_vendor` не показувати навіть якщо кнопка `MAC` активна.

Причина:
- vendor для random / locally administered MAC може вводити в оману
- `mac_vendor` у такому випадку не є надійною ознакою виробника

---

## State management

Необхідно додати стан UI:

```text
showMacVendor: boolean
```

Рекомендоване значення за замовчуванням:

```text
true
```

Щоб після впровадження TASK-103 поточна поведінка залишилась видимою.

Якщо в UI вже є механізм збереження налаштувань користувача у localStorage, можна зберігати стан кнопки `MAC`.

Якщо такого механізму немає - localStorage не додавати в межах цієї задачі без необхідності.

Необхідно додати або оновити UI тести:
