# TASK-075 - UI refinement: Align current layout to target design (single-line structured toolbar + cleaner device rows)

## Контекст

Користувач надав два стани UI:

1. Поточний вигляд (перевантажений, нерівномірний, слабка ієрархія)
2. Бажаний вигляд (чистий, структурований, читабельний)

Задача — привести UI до цільового вигляду.

---

## Мета задачі

- зробити toolbar компактним і логічним
- покращити читабельність рядків пристроїв
- уніфікувати стилі
- прибрати “візуальний шум”

---

# 1. Toolbar — фінальна структура

## Цільовий вигляд

```
[ FILTERS ]  Clear ✕   |   [ End | All ]   Devices: X | 🟢x 🟡x 🟣x 🔴x   |   ⇅ ⟳   Auto [ON]
```

---

## Вимоги

### 1.1 Один рядок
- всі елементи в один горизонтальний ряд
- без переносу

---

### 1.2 Чіткі блоки

#### Ліво:
```
[ FILTERS ] Clear ✕
```

#### Центр:
```
[ End | All ]   Devices: ...
```

#### Право:
```
⇅ ⟳   Auto [ON]
```

---

### 1.3 Filters

- використовувати badge з таблиці
- однакові кольори
- clickable
- optional: removable ✕

---

### 1.4 Mode

```
[ End | All ]
```

- segmented control
- default: End
- без dropdown

---

### 1.5 Devices summary

```
Devices: X | 🟢x 🟡x 🟣x 🔴x
```

ВАЖЛИВО:
- залежить тільки від Mode
- НЕ залежить від filters

---

### 1.6 Actions

- ⇅ Sort
- ⟳ Refresh
- Auto toggle

---

# 2. Рядок пристрою (Device row)

## Поточна проблема

- інформація розкидана
- важко сканувати
- багато “шуму”

---

## Цільовий вигляд

### Один компактний блок:

```
IPHONE-okudr (192.168.101.185)
20:37:A5:87:2A:13
[ BADGES ]    ● Online   02:14
```

---

## Вимоги

### 2.1 Об’єднання інформації

Замість:

- MAC
- IP
- hostname

Зробити:

```
hostname (IP)
MAC
```

---

### 2.2 Status + Timer

Об’єднати:

```
● Online   02:14
```

- кольорова точка
- таймер поруч

---

### 2.3 Badges

- STATIC / RANDOM / etc
- один стиль всюди

---

### 2.4 Stale дані (з TASK-074)

Якщо дані збережені:

```
192.168.101.12 [STALE]
```

або:
- сірий текст
- або badge

---

# 3. Візуальна ієрархія

## Пріоритет:

1. Статус (online/offline)
2. hostname
3. IP
4. MAC
5. badges

# 4. Вирівнювання
